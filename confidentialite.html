2. Données personnelles collectées

Nous collectons uniquement les données strictement nécessaires à la gestion de vos commandes :

  • Identité : nom, prénom (lors du paiement Stripe)
  • Coordonnées : adresse email, adresse de livraison, numéro de téléphone
  • Données de paiement : aucune donnée bancaire n'est stockée sur nos serveurs. Le paiement est intégralement géré par Stripe Payments Europe, Ltd., prestataire certifié PCI-DSS niveau 1
  • Données de commande : historique d'achats, montants, dates
  • Données techniques : contenu du panier (stocké uniquement dans votre navigateur via localStorage)

3. Finalités du traitement

Vos données sont utilisées pour :

  • Traiter et livrer vos commandes
  • Gérer la relation client (réponse aux questions, service après-vente)
  • Émettre des factures et respecter nos obligations comptables et fiscales
  • Lutter contre la fraude (vérification de paiement via Stripe)

4. Base légale du traitement

Le traitement de vos données repose sur :

  • L'exécution du contrat de vente (article 6.1.b RGPD) pour la gestion de votre commande
  • Le respect d'obligations légales (article 6.1.c RGPD) pour la conservation comptable
  • L'intérêt légitime (article 6.1.f RGPD) pour la prévention de la fraude

5. Destinataires des données

Vos données ne sont jamais vendues ni cédées à des tiers à des fins commerciales. Elles peuvent être transmises uniquement à :

  • Stripe Payments Europe, Ltd. (sous-traitant paiement) — Politique Stripe
  • Vercel Inc. (hébergement technique du site) — Politique Vercel
  • Les transporteurs choisis pour la livraison de votre commande
  • Les administrations légalement habilitées en cas de réquisition (autorités fiscales, judiciaires)

6. Durée de conservation

  • Données de commande et facturation : 10 ans (obligation légale comptable - article L123-22 du Code de commerce)
  • Données de compte client : 3 ans à compter de la dernière activité
  • Données de paiement : conservées par Stripe selon leur propre politique (jamais stockées chez nous)
  • Données du panier (localStorage) : stockées uniquement sur votre appareil, supprimables à tout moment

7. Transferts hors UE

Notre hébergeur Vercel Inc. est basé aux États-Unis. Les transferts de données sont encadrés par les Clauses Contractuelles Types de la Commission Européenne et le Data Privacy Framework auquel Vercel adhère.
